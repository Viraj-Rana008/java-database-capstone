# User Story Template

## Admin User Stories

As an Admin, ...
- I can login into admin dashboard to manage the platform.
- I can logout from the system to avoid unwanted access
- I can add Doctors to the platform to be available for the patients.
- I can remove doctors from the platform who re not available.
- I can run a procedure to fetch the number of appointments in a month to maintain usage statistics.


## Patient User Stories

As a Patient, ...
- I can view list of doctors without logging-in to explore available doctors
- I can register on the platform to create an account
- I can login into the account to manage my bookings
- I can logout from the account to keep my account secure
- I can book an appointment with a doctor
- I can view my booked appointments


## Doctor User Stories

As a doctor,
- I can register on the platform to create my account
- I can login into the account to manage my appointments
- I can logout from the account to secure it
- I can view my appointments from calendar to stay organised
- I can cancel an appointment to handle unavailability
- I can mark unavailability in the calendar to avoid taking bookings for that period
- I can view patient's details to come prepared before appointment
- I can update my profile with qualification and contact information so that patient have the up-to-date information
